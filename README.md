# Pin & Tell – Admin Management System

Pin & Tell is a web-based platform with a dedicated **Admin Management System** used to control, monitor, and manage the entire application.

This website is intended **for administrators only** and is not the public user-facing version of Pin & Tell.

## ⚙️ Main Features

- Secure admin login
- Dashboard overview and statistics
- Manage users
- Manage reports / pins / submitted content
- Review uploaded media
- Activity logs
- Update platform settings
- Manage Terms of Service and Privacy Policy
- Profile and password management

## 🔒 Security

- Session-based authentication
- Role-based access control
- Password protection
- Input validation
- Activity tracking

## 🧰 Technologies Used

- PHP
- MySQL / MariaDB
- HTML
- CSS
- JavaScript

## 📁 Project Structure

- `/css` – Stylesheets
- `/js` – Scripts
- `/images` – Uploaded files and assets
- `/connection` – Database connection files
- `/includes` – Reusable PHP components

## 🚀 Notes

This repository contains the **admin management website** for Pin & Tell.  
All tools and pages inside this project are focused on administration and moderation functions.
